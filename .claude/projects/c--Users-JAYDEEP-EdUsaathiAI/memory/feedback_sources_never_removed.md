Sources are never removed, only added.
PubMed, RCSB, Indian Kanoon, Wolfram, NIST, NASA, ChemSpider, PubChem, OpenAnatomy, ISRO Bhuvan, ScienceDirect, Scopus — all stay in every future build.

**Why:** The student always has access to every source. Removing a tab removes a research capability a student may depend on.

**How to apply:** New sources are added alongside existing ones. If a tab bar gets crowded — group them under a "Literature" or "Data" category tab. Never delete a source tab. Never replace one source with another. Only add.
